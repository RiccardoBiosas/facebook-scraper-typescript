## About

Command line tool to scrape likes and followers from a facebook page. The tool leverages [tesseract.js](https://github.com/naptha/tesseract.js?files=1) (optical character recognition), [puppeteer](https://github.com/puppeteer/puppeteer) and regular expressions to extract information from a target page in a "DOM-agnostic" fashion.
